# Security Policy

## Reporting Vulnerabilities

Report vulnerabilities via [GitHub Security Advisories](../../security/advisories/new).

Do not open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | Yes       |

## Scope

All modules in this repository are in scope for security reports:

- `src/` — React frontend components
- `electron/` — Electron main process, proxy server, analyzers
- `electron/proxy/` — API proxy and SSE interception
- `electron/db/` — SQLite database layer
